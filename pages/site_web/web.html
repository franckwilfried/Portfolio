• 2023Site web - Elior restauration

    Entreprise: Elior.
    Description: Site web vitrine.
    Toujours animé par l’objectif de développer mes compétences, j’ai profité de ma période de job étudiant chez Elior Restauration pour me former en autodidacte aux langages HTML, CSS et JavaScript. Et quoi de mieux qu’un projet concret pour apprendre efficacement ?
    J’ai donc conçu et développé ce site web qui :

    1. Met en place un menu déroulant pour naviguer
    2. Utilise un formulaire de contact
    3. Presente les produits du restaurant avec les prix
    4. Inclus un panier, pour voir la liste et coût d'achat
    5. Inclus des calculs Javascript pour les coûts
    6. Utilise le localStorage pour tenir à jour le panier
